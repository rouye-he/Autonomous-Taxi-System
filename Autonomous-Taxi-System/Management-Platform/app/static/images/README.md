# 城市地图背景图片使用说明

## 文件命名规范

城市地图背景图片应遵循以下命名规范：

1. 使用城市拼音全拼命名，全部小写
2. 格式为：`城市拼音.png`，例如：`shenyang.png`、`beijing.png`、`shanghai.png`

## 支持的城市列表

系统目前支持以下城市的地图背景：

- 沈阳市 (shenyang.png)
- 北京市 (beijing.png)
- 上海市 (shanghai.png)
- 广州市 (guangzhou.png)
- 深圳市 (shenzhen.png)
- 杭州市 (hangzhou.png)
- 南京市 (nanjing.png)
- 成都市 (chengdu.png)
- 重庆市 (chongqing.png)
- 武汉市 (wuhan.png)
- 西安市 (xian.png)

## 图片要求

1. 分辨率：1000×1000像素
2. 格式：PNG格式，支持透明度
3. 内容：城市的卫星图或规划图，最好有明显的道路和地标
4. 大小：建议文件大小控制在5MB以内，以保证加载速度

## 添加新城市图片步骤

1. 准备符合要求的城市地图图片
2. 按照命名规范重命名图片
3. 将图片放入`app/static/images/`目录
4. 重启应用后，在地图视图中即可选择新增的城市

## 注意事项

1. 如果某个城市没有对应的背景图片，系统会自动使用默认的沈阳地图作为背景
2. 修改或添加图片后无需修改代码，系统会自动检测并加载对应城市的背景图片
3. 为保证最佳显示效果，建议所有城市地图使用相同的比例尺和风格
